# SKSTU Logo

Brand assets for the SKSTU HeLuo model architecture concept.

- `skstu_logo.svg`: clean horizontal lockup for documents, website headers, and product pages.
- `skstu_mark.svg`: simplified primary brand mark for app surfaces, avatars, and navigation.
- `skstu_favicon.svg`: minimal S/K monogram for favicon and very small icon sizes.
- `skstu_detail.svg`: detailed concept mark for large brand presentations.
- `skstu_logo.html`: local preview page for comparing the assets.

Design logic:

- Luo Shu is kept as a hidden 3x3 inference matrix.
- He Tu is reduced to orbit points in production marks, while pair labels remain in the detailed presentation mark.
- SKSTU is led by the central S/K monogram: the S is a model stream, and the K is a routing branch.
